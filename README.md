# **Projeto Integrador - Programador Web**

Este repositório contém o desenvolvimento do meu **Projeto Integrador de Conclusão do Curso de Programador Web**.

O projeto consiste na criação de uma **Landing Page institucional** integrada a um **sistema de gerenciamento (CRUD)** para administração de conteúdos, utilizando as tecnologias **HTML5, CSS3, PHP e MySQL**.

O objetivo é aplicar na prática os conhecimentos adquiridos ao longo do curso, desenvolvendo uma aplicação web funcional, responsiva e organizada, seguindo boas práticas de estruturação de código e desenvolvimento web.
